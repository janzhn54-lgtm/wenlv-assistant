# UrbanDesignAI - AI城市规划设计助手

> 输入用地红线图 + 规划条件 → AI自动生成完整设计方案

## 🎯 项目概述

这是一个AI驱动的城市规划设计软件，专为爸爸（文旅策划从业者）打造。

### 核心功能

✅ **智能读取CAD** - 自动识别用地红线图  
✅ **AI方案生成** - 根据规划指标自动生成布局  
✅ **渲染图输出** - 彩色平面图、鸟瞰效果图  
✅ **设计文档** - 完整的设计逻辑和理念说明  
✅ **学习进化** - 持续学习优秀案例，越用越聪明

---

## 📂 项目结构

```
UrbanDesignAI/
├── backend/                    # 后端核心模块
│   ├── cad_handler.py         # CAD处理模块
│   ├── ai_design_engine.py    # AI设计生成引擎
│   └── render_engine.py       # 渲染图生成引擎
├── frontend/                   # 前端界面
│   └── index.html            # Web操作界面
├── outputs/                    # 输出成果

│   ├── analysis/             # 分析图
│   └── design_document.md    # 设计说明文档
├── main.py                    # 主程序入口
└── README.md                  # 项目说明

```

## 🚀 快速开始

### 安装依赖

```bash
pip install ezdxf matplotlib numpy
```

### 运行程序

```bash
python3 main.py
```

按提示输入规划参数，AI自动生成设计方案。

---

## 📋 输入参数

| 参数 | 说明 | 示例 |
|------|------|------|
| 功能定位 | 项目类型 | 文旅小镇/住宅/商业 |
| 容积率 | 总建筑面积/用地面积 | 1.5 |
| 建筑密度 | 建筑基底面积占比 | 35% |
| 绿化率 | 绿地面积占比 | 30% |
| 限高 | 建筑最大高度 | 24米 |

## 💡 使用示例

### 示例1：文旅小镇

```python
constraints = {
    'function_type': '文旅小镇',
    'plot_ratio': 1.5,
    'building_density': 35.0,
    'green_rate': 30.0,
    'height_limit': 24.0
}
```

生成功能分区：
- 核心文化展示区 15%
- 商业休闲区 25%
- 住宿度假区 30%
- 景观游憩区 20%
- 配套服务区 10%

```python
constraints = {

}
```
