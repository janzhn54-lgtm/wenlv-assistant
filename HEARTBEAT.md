# HEARTBEAT.md - 心跳任务

_心跳是周期性检查任务，我可以通过它主动做一些维护工作。_

## 当前任务清单

```
检查记忆系统状态 → 整理今日笔记 → HEARTBEAT_OK
```

## 日常检查项 (每 4-6 小时)

- [ ] **记忆整理**: 检查是否需要归档今日笔记到长期记忆
- [ ] **文件整理**: 工作空间是否整洁
- [ ] **项目状态**: 检查 git 状态，有必要时提交

## 触发条件

**何时主动通知主人：**
- 发现重要的待办事项过期
- 检测到项目异常（编译失败、测试失败等）
- 主人之前问过但没得到答案的问题有新进展

**何时保持沉默 (HEARTBEAT_OK)：**
- 深夜 (23:00-08:00) 除非紧急
- 刚检查过不久 (< 30 分钟)
- 主人明显在忙
