# 小白设计工作室 - 完整工作流与工具链体系

## 🎯 目标
建立对标国内一流规划院/设计公司的专业工作体系

---

## 一、工作流程优化（对标优秀团队）

### 参考对象
- **中规院/同济院**：严谨的规划流程
- **GOA大象设计**：文旅项目经验
- **Sasaki**：城市设计方法论
- **MAD**：创新设计流程

### 优化后的6阶段工作流

```
┌─────────────────────────────────────────────────────────────┐
│  阶段0: 项目启动 → 阶段1: 调研分析 → 阶段2: 概念设计          │
│      ↓                                               ↓       │
│  阶段5: 交付服务 ← 阶段4: 深化设计 ← 阶段3: 方案设计          │
└─────────────────────────────────────────────────────────────┘
```

---

### 阶段0: 项目启动（0.5天）

**输入**: 爸爸的需求简报
**输出**: 《项目任务书》

**工作内容**:
1. 需求澄清（3个问题确认法）
2. 资料收集清单
3. 工作计划排期
4. 团队分工确认

**交付物**:
- 项目任务书（1页）
- 资料需求清单
- 甘特图进度表

---

### 阶段1: 调研分析（2-3天）

**输入**: 爸爸提供的资料 + 自行调研
**输出**: 《调研分析报告》

**参与Agent**: 规划法规专家 + 投资金融专家 + 文旅运营专家

**工作内容**:

#### 1.1 规划法规分析（规划法规专家主导）
- 上位规划解读
- 规划条件梳理
- 限制因素识别
- 审批路径规划

**参考团队**: 中规院规划所
**交付物**: 
- 规划条件分析表
- 限制因素分析图
- 法规合规性评估

#### 1.2 市场分析（文旅运营专家主导）
- 区位分析
- 竞品项目研究（重点分析运营模式、客群定位）
- 客群定位（消费心理、行为模式深度分析）
- SWOT分析
- 运营前瞻性评估

**参考团队**: 世联行/戴德梁行
**交付物**:
- 区位分析图
- 竞品案例库（含运营模式拆解）
- 客群画像报告
- 市场定位报告

#### 1.3 投资可行性初判（投资金融专家主导）
- 投资规模估算
- 收益来源分析
- 初步财务指标测算
- 投资风险识别
- 融资路径预判

**交付物**:
- 投资可行性初步评估
- 收益结构预测
- 风险提示清单

#### 1.4 基地分析（规划法规专家主导）
- 现状测绘分析
- 交通可达性
- 景观资源评估
- 基础设施评估

**参考团队**: Sasaki
**交付物**:
- 基地分析图集
- GIS分析图
- 现状照片集

---

### 阶段2: 概念设计（3-5天）

**输入**: 调研报告
**输出**: 《概念设计方案》（多方案比选）

**参与Agent**: 城市设计专家 + 投资金融专家 + 文旅运营专家 + 建筑设计专家

**工作内容**:

#### 2.1 投资测算深化（投资金融专家主导）
- 详细投资估算（土地、建安、前期、财务费用）
- 收益预测模型（分业态、分年度）
- 财务指标计算（NPV、IRR、回收期）
- 敏感性分析
- 融资方案设计（股权、债权、REITs等）

**交付物**:
- 投资测算表（Excel模型）
- 收益预测报告
- 融资方案建议书

#### 2.2 运营策划前置（文旅运营专家主导）
- 商业定位与业态策划
- 客群细分与产品匹配
- 运营模式设计（自营/租赁/合作）
- 收入结构规划
- 关键运营指标设定

**交付物**:
- 运营策划方案
- 业态配比建议
- 收入结构预测

#### 2.3 概念生成（城市设计专家主导）
- 案例研究（国内外，重点分析运营成功案例）
- 设计理念提炼
- 主题定位
- 功能策划（与运营策划联动）

**参考团队**: GOA/隈研吾事务所
**交付物**:
- 案例研究报告
- 设计理念阐述
- 功能配比表

#### 2.4 空间规划（城市设计专家主导）
- 总体布局（3-5个方案）
- 功能分区（与业态规划联动）
- 交通组织
- 景观结构

**参考团队**: BIG/Sasaki
**交付物**:
- 总平面图（3-5套）
- 功能分区图
- 交通分析图
- 空间结构图

#### 2.5 方案比选（投资金融专家+文旅运营专家参与评审）
- 多方案对比分析
- 投资收益对比
- 运营可行性评估
- 优缺点评估
- 推荐方案

**交付物**:
- 方案比选表（含投资、运营维度）
- 推荐方案说明

---

### 阶段3: 方案设计（5-7天）

**输入**: 选定的概念方案
**输出**: 《方案设计成果》

**参与Agent**: 城市设计专家 + 建筑设计专家 + 投资金融专家 + 文旅运营专家

**工作内容**:

#### 3.1 详细投资测算（投资金融专家主导）
- 分年度投资计划
- 现金流预测
- 融资方案细化
- 退出路径设计（IPO/REITs/并购）
- 投资回报分析

**交付物**:
- 详细投资测算表
- 现金流预测表
- 融资方案书
- 投资回报分析

#### 3.2 运营策划深化（文旅运营专家主导）
- 招商策略（业态规划、品牌组合、租金策略）
- 运营机制（组织架构、服务流程、考核体系）
- 客户体验设计（动线、停留点、消费触发）
- 年度活动计划
- 会员体系设计

**交付物**:
- 招商策划方案
- 运营机制手册
- 客户体验设计
- 年度活动计划表

#### 3.3 详细规划（城市设计专家主导）
- 总平面深化
- 道路竖向设计
- 管线综合规划
- 指标核算

**参考团队**: 同济院/浙江省院
**交付物**:
- 详细总平面图
- 竖向设计图
- 管线综合图
- 经济技术指标表

#### 3.4 建筑设计（建筑设计专家主导）
- 建筑单体方案
- 立面设计
- 剖面设计
- 典型平面

**参考团队**: MAD/GOA
**交付物**:
- 建筑平面图
- 立面图
- 剖面图
- 效果图

#### 3.5 景观设计（如需要）
- 景观总平面图
- 重要节点设计
- 植物配置
- 小品设计

**交付物**:
- 景观平面图
- 节点效果图
- 植物配置表

---

### 阶段4: 深化设计（7-14天）

**输入**: 确定的方案
**输出**: 《深化设计成果》（施工图深度）

**参与Agent**: 全体专家

**工作内容**:

#### 4.1 规划深化（规划法规专家主导）
- 规划说明书
- 规划图纸（CAD）
- 规划模型（如需）

**交付物**:
- 规划设计说明书
- CAD图纸全套
- 规划模型文件

#### 4.2 投资深化报告（投资金融专家主导）
- 《可行性研究报告》（符合国家发改委格式）
- 《投资测算书》（详细财务模型）
- 《融资方案建议书》
- 《风险评估报告》

**格式标准**: 国家发改委《建设项目可行性研究报告编制指南》
**交付物**:
- 可行性研究报告（完整版）
- 投资测算Excel模型
- 融资方案书
- 风险评估报告

#### 4.3 运营策划手册（文旅运营专家主导）
- 《招商手册》（业态、品牌、政策）
- 《运营管理手册》（流程、标准、考核）
- 《开业筹备计划》（倒排工期、关键节点）
- 《年度运营计划》（活动、营销、目标）

**交付物**:
- 招商手册
- 运营管理手册
- 开业筹备计划
- 年度运营计划

#### 4.4 专项研究
- 交通专项（交通影响评价）
- 景观专项
- 建筑专项
- 市政专项
- **投资专项**（如有REITs等特殊融资需求）
- **运营专项**（如有特殊业态需论证）

**交付物**:
- 各专项研究报告

---

### 阶段5: 交付服务（持续）

**输入**: 全套成果
**输出**: 汇报/审查/实施支持

**参与Agent**: 根据需求灵活调配

**工作内容**:

#### 5.1 汇报支持
- PPT制作
- 汇报演讲稿
- Q&A准备
- **投资收益汇报**（投资金融专家支持）
- **运营前景汇报**（文旅运营专家支持）

#### 5.2 审查配合
- 回复审查意见
- 方案调整
- 补充材料
- **融资审批配合**（投资金融专家支持）
- **招商前置启动**（文旅运营专家支持）

#### 5.3 实施跟踪
- 设计交底
- 现场配合
- 变更处理
- **融资落地支持**（投资金融专家）
- **招商执行支持**（文旅运营专家）
- **运营筹备跟踪**（文旅运营专家）

- **文旅运营专家**: 跟踪项目开业运营，收集反馈，持续优化运营方案
- 定期向爸爸汇报项目运营数据
- 总结经验教训，更新案例库

## 二、软件工具链设计

### 核心原则
1. **服务器可运行**（Linux环境）
2. **免费/开源**（避免版权问题）
3. **专业级输出**（不输Adobe）
4. **自动化能力**（可脚本批量处理）

### 替代方案矩阵

| Adobe软件 | 替代方案 | 服务器可行性 | 输出质量 |
|-----------|----------|--------------|----------|
| Photoshop | GIMP + ImageMagick | ✅ 完美 | ⭐⭐⭐⭐ |
| Illustrator | Inkscape | ✅ 完美 | ⭐⭐⭐⭐ |
| InDesign | Scribus | ✅ 完美 | ⭐⭐⭐ |
| Premiere | FFmpeg | ✅ 完美 | ⭐⭐⭐⭐ |
| After Effects | Blender | ✅ 可用 | ⭐⭐⭐ |
| Acrobat | LibreOffice + PDFtk | ✅ 完美 | ⭐⭐⭐⭐ |
| Lightroom | darktable | ✅ 完美 | ⭐⭐⭐⭐ |

### 推荐工具链配置

#### 图像处理
```bash
# 安装命令
sudo apt-get install -y \
    gimp \
    imagemagick \
    inkscape \
    scribus \
    darktable \
    blender
```

**GIMP**（替代Photoshop）
- 图层、蒙版、滤镜齐全
- 支持PSD格式
- Python-Fu脚本自动化

**ImageMagick**（批量处理神器）
```bash
# 批量调整大小
convert *.jpg -resize 1920x1080 output/

# 添加水印
composite -gravity southeast watermark.png input.jpg output.jpg

# 生成PDF
convert *.jpg output.pdf
```

#### 矢量绘图
**Inkscape**（替代Illustrator）
- SVG原生支持
- 精确的路径编辑
- 位图追踪功能

**Python + Matplotlib**（程序化出图）
```python
# 分析图自动生成
import matplotlib.pyplot as plt
import numpy as np

# 饼图、柱状图、热力图
plt.pie(data, labels=labels, autopct='%1.1f%%')
plt.savefig('analysis_chart.png', dpi=300)
```

#### 3D渲染
**Blender**（替代3ds Max/C4D）
- 专业级渲染引擎
- Python API自动化
- 支持CAD导入

```python
# 程序化生成建筑
import bpy

# 创建建筑体块
bpy.ops.mesh.primitive_cube_add(size=10)
```

#### 排版出版
**Scribus**（替代InDesign）
- 专业桌面出版
- PDF/X-1a标准支持
- 模板系统

**LaTeX**（学术级排版）
```latex
\documentclass{article}
\usepackage{graphicx}
\begin{document}
\title{可行性研究报告}
\maketitle
\end{document}
```

#### CAD制图
**LibreCAD**（替代AutoCAD）
- 2D CAD制图
- DXF/DWG支持
- 命令行自动化

**Python + ezdxf**（程序化CAD）
```python
import ezdxf

doc = ezdxf.new('R2010')
msp = doc.modelspace()

# 绘制用地红线
msp.add_lwpolyline(points, close=True)

# 添加标注
msp.add_text("酒店", height=2.5).set_pos((x, y))

doc.saveas('plan.dxf')
```

#### GIS分析
**QGIS**（替代ArcGIS）
- 完整GIS功能
- Python插件开发
- 丰富的分析工具

**Python + GeoPandas**
```python
import geopandas as gpd

# 读取地块数据
gdf = gpd.read_file('land.shp')

# 分析缓冲区
buffer = gdf.buffer(distance=100)

# 生成分析图
gdf.plot(column='land_use', cmap='viridis')
plt.savefig('gis_analysis.png', dpi=300)
```

## 三、输出文件规范

### 3.1 可行性研究报告

**参考标准**: 
- 《建设项目可行性研究报告编制指南》
- 国家发改委格式要求

**内容结构**:
```
第一章 总论
  1.1 项目背景
  1.2 编制依据
  1.3 研究范围
  1.4 主要结论

第二章 项目建设必要性
  2.1 政策背景
  2.2 市场需求
  2.3 社会效益

第三章 市场分析
  3.1 宏观环境
  3.2 区域市场
  3.3 竞争分析
  3.4 客群定位

第四章 建设方案
  4.1 项目选址
  4.2 建设规模
  4.3 功能定位
  4.4 规划设计

第五章 投资估算
  5.1 投资构成
  5.2 资金筹措
  5.3 财务分析

第六章 风险分析
  6.1 政策风险
  6.2 市场风险
  6.3 应对措施

第七章 结论与建议
```

**格式要求**:
- 纸张：A4
- 字体：正文宋体小四，标题黑体
- 页边距：上下2.54cm，左右3.17cm
- 行距：1.5倍
- 页码：底部居中
- 封面：项目名称+编制单位+日期

---

- 世联行/戴德梁行报告格式

```
一、宏观市场分析
  1.1 政策环境

三、项目定位分析
  3.1 SWOT分析

四、案例研究
  4.1 成功案例

五、结论建议
```

### 3.3 规划设计文件

#### 规划说明书

**章节结构**:
```
第一章 规划背景
第二章 现状分析
第三章 规划理念
第四章 总体布局
第五章 专项规划
```

#### 规划图纸

**必须包含的图纸（8张基础图）**:
1. 区位分析图
2. 土地利用现状图
3. 总体规划平面图
4. 功能分区图
5. 道路交通规划图
6. 绿地景观规划图
7. 竖向规划图
8. 效果图（鸟瞰+人视）

**图纸规范**:
- 比例：1:500 / 1:1000 / 1:2000
- 坐标系：2000国家大地坐标系
- 图例：规范图例库
- 指北针：统一方向
- 比例尺：图示+数字双标注

**CAD制图标准**:
- 图层命名：GB/T 50001
- 线型线宽：区分建筑/道路/绿化
- 字体：仿宋/宋体
- 标注：精确到小数点后2位

### 3.4 效果图规范

**类型与数量**:
- 鸟瞰图：1-2张（白天+夜景）
- 人视图：3-5张（重要节点）
- 平面图：1张（彩色总平）
- 分析图：若干
- **运营场景图**：2-3张（人流、活动、业态）

**技术要求**:
- 分辨率：300dpi以上
- 尺寸：A1图幅（594×841mm）
- 格式：JPG/PNG + PSD源文件
- 风格：统一调性

**制作软件**:
- 建模：Blender / SketchUp
- 渲染：Blender Cycles / Lumion
- 后期：GIMP

**参考标准**:

**内容结构**:
```
第一章 投资估算
  1.1 土地成本
  1.2 建安工程费
  1.3 前期工程费
  1.4 基础设施费
  1.5 开发间接费
  1.6 财务费用
  1.7 不可预见费

第二章 资金筹措
  2.1 资本金
  2.2 银行贷款
```

### 4.1 批量处理脚本

**批量生成分析图**:
```python
#!/usr/bin/env python3
"""批量生成规划分析图"""

import matplotlib.pyplot as plt
import pandas as pd

def generate_land_use_chart(data, output_path):
    """生成用地平衡图"""
    fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 6))
    
    # 饼图
    ax1.pie(data['area'], labels=data['type'], autopct='%1.1f%%')
    ax1.set_title('用地面积占比')
    
    # 柱状图
    ax2.bar(data['type'], data['area'])
    ax2.set_title('用地面积对比')
    ax2.set_ylabel('面积(㎡)')
    
    plt.tight_layout()
    plt.savefig(output_path, dpi=300)
    plt.close()

# 使用示例
data = pd.DataFrame({
    'type': ['酒店', '住宅', '商业', '绿化'],
    'area': [25000, 40000, 15000, 20000]
})
generate_land_use_chart(data, 'land_use_analysis.png')
```
