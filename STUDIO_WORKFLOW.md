# 小白设计工作室 - 完整工作流与工具链体系

## 🎯 目标
建立对标国内一流规划院/设计公司的专业工作体系

---

## 一、工作流程优化（对标优秀团队）

### 参考对象
- **中规院/同济院**：严谨的规划流程
- **GOA大象设计**：文旅项目经验
- **Sasaki**：城市设计方法论
- **MAD**：创新设计流程

### 优化后的6阶段工作流

```
┌─────────────────────────────────────────────────────────────┐
│  阶段0: 项目启动 → 阶段1: 调研分析 → 阶段2: 概念设计          │
│      ↓                                               ↓       │
│  阶段5: 交付服务 ← 阶段4: 深化设计 ← 阶段3: 方案设计          │
└─────────────────────────────────────────────────────────────┘
```

---

### 阶段0: 项目启动（0.5天）

**输入**: 爸爸的需求简报
**输出**: 《项目任务书》

**工作内容**:
1. 需求澄清（3个问题确认法）
2. 资料收集清单
3. 工作计划排期
4. 团队分工确认

**交付物**:
- 项目任务书（1页）
- 资料需求清单
- 甘特图进度表

---

### 阶段1: 调研分析（2-3天）

**输入**: 爸爸提供的资料 + 自行调研
**输出**: 《调研分析报告》+《市场调研报告》+《文化资源调查报告》

**参与Agent**: 规划法规专家 + 投资金融专家 + 文旅运营专家 + **文化策划专家**

**工作内容**:

#### 1.1 规划法规分析（规划法规专家主导）
- 上位规划解读
- 规划条件梳理
- 限制因素识别
- 审批路径规划

**参考团队**: 中规院规划所
**交付物**: 
- 规划条件分析表
- 限制因素分析图
- 法规合规性评估

#### 1.2 市场分析 + 市场调研报告（文旅运营专家主导）
- **区位分析**
- **竞品项目深度研究**（运营模式、客群定位、收益结构）
- **客群定位**（消费心理、行为模式深度分析）
- **SWOT分析**
- **运营前瞻性评估**

**新增交付物 - 《市场调研报告》**: 
```
一、宏观市场环境
  1.1 政策环境分析
  1.2 经济环境影响
  1.3 社会文化趋势
  1.4 技术变革影响

二、行业市场分析
  2.1 文旅行业市场规模与增长
  2.2 细分市场分析（酒店/景区/商业）
  2.3 行业发展趋势
  2.4 新兴业态研究

三、区域市场研究
  3.1 绍兴市文旅市场概况
  3.2 越城区/迪荡湖板块分析
  3.3 供需关系分析
  3.4 价格水平研究

四、竞品项目调研
  4.1 直接竞品分析（3-5个）
    - 项目概况
    - 产品定位
    - 运营模式
    - 收益结构（如可获取）
    - 客群特征
    - 优劣势分析
  4.2 间接竞品分析
  4.3 标杆案例研究

五、客群需求调研
  5.1 目标客群画像
  5.2 消费行为分析
  5.3 需求痛点识别
  5.4 支付意愿研究

六、市场机会与风险评估
  6.1 市场机会识别
  6.2 市场风险评估
  6.3 进入策略建议

七、结论与建议
```

**参考团队**: 世联行/戴德梁行/艾瑞咨询
**交付物**:
- 《市场调研报告》（完整版，30-50页）
- 区位分析图
- 竞品对比分析表
- 客群画像图谱
- 市场定位建议

#### 1.3 投资可行性初判（投资金融专家主导）
- 投资规模估算
- 收益来源分析
- 初步财务指标测算
- 投资风险识别
- 融资路径预判

**交付物**:
- 投资可行性初步评估
- 收益结构预测
- 风险提示清单

#### 1.4 基地分析（规划法规专家主导）
- 现状测绘分析
- 交通可达性
- 景观资源评估
- 基础设施评估

**参考团队**: Sasaki
**交付物**:
- 基地分析图集
- GIS分析图
- 现状照片集

---

#### 1.5 文化资源调研（文化策划专家主导）
- **地域文化研究**: 历史沿革、文化脉络、精神内核
- **物质文化调查**: 古建筑、历史遗迹、文化景观
- **非物质文化挖掘**: 民俗、技艺、节庆、传说
- **周边文化资源**: 项目周边文化景点、文化设施
- **文化价值评估**: 独特性、影响力、活化潜力

**新增交付物 - 《文化资源调查报告》**:
```
第一章 区域文化背景
  1.1 历史文化沿革
  1.2 地域文化特征
  1.3 文化精神内核

第二章 物质文化资源
  2.1 历史建筑遗存
  2.2 文化景观资源
  2.3 工业遗产资源（如适用）
  2.4 保护状态评估

第三章 非物质文化资源
  3.1 民俗文化
  3.2 传统技艺
  3.3 地方戏曲/曲艺
  3.4 节庆活动
  3.5 民间传说/故事

第四章 周边文化环境
  4.1 周边文化景点
  4.2 文化设施分布
  4.3 文化产业发展
  4.4 文化氛围评估

第五章 文化价值评估
  5.1 独特性评估
  5.2 影响力评估
  5.3 活化潜力评估
  5.4 与项目契合度分析

第六章 可利用文化元素
  6.1 核心文化符号
  6.2 文化故事素材
  6.3 文化体验素材
  6.4 IP开发建议

第七章 结论与建议
  7.1 文化定位建议
  7.2 文化融入策略
  7.3 文化业态建议
```

**参考团队**: 清华同衡/巅峰智业文化团队
**交付物**:
- 《文化资源调查报告》（完整版，20-30页）
- 文化资源分布图
- 文化价值评估表
- 可利用文化元素清单
- 文化定位初步建议

---

### 阶段2: 概念设计（3-5天）

**输入**: 调研报告（含文化资源调查报告）
**输出**: 《概念设计方案》+《文化策划方案》

**参与Agent**: 城市设计专家 + 投资金融专家 + 文旅运营专家 + 建筑设计专家 + **文化策划专家**

**工作内容**:

#### 2.1 投资测算深化（投资金融专家主导）
- 详细投资估算（土地、建安、前期、财务费用）
- 收益预测模型（分业态、分年度）
- 财务指标计算（NPV、IRR、回收期）
- 敏感性分析
- 融资方案设计（股权、债权、REITs等）

**交付物**:
- 投资测算表（Excel模型）
- 收益预测报告
- 融资方案建议书

#### 2.2 运营策划前置（文旅运营专家主导）
- 商业定位与业态策划
- 客群细分与产品匹配
- 运营模式设计（自营/租赁/合作）
- 收入结构规划
- 关键运营指标设定

**交付物**:
- 运营策划方案
- 业态配比建议
- 收入结构预测

#### 2.3 概念生成（城市设计专家主导）
- 案例研究（国内外，重点分析运营成功案例）
- 设计理念提炼
- 主题定位
- 功能策划（与运营策划联动）

**参考团队**: GOA/隈研吾事务所
**交付物**:
- 案例研究报告
- 设计理念阐述
- 功能配比表

#### 2.4 空间规划（城市设计专家主导，文化策划专家配合）
- 总体布局（3-5个方案）
- 功能分区（与业态规划、文化策划联动）
- 交通组织
- 景观结构
- **文化空间序列**（文化策划专家：文化轴线、场所精神、文化节点）

**参考团队**: BIG/Sasaki
**交付物**:
- 总平面图（3-5套）
- 功能分区图
- 交通分析图
- 空间结构图
- **文化空间规划图**

#### 2.5 文化策划融入设计（文化策划专家主导）
- **文化主题提炼**: 基于调研确定核心文化主题
- **文化IP策划**: IP形象、故事线、视觉系统
- **文化融入策略**: 如何在空间、建筑、景观中表达文化
- **文化体验设计**: 文化动线、体验节点、互动方式
- **文化业态建议**: 文化展示、演艺、商业、活动

**新增交付物 - 《文化策划方案》**:
```
第一章 文化主题定位
  1.1 核心文化主题
  1.2 主题演绎框架
  1.3 文化定位语
  1.4 文化愿景

第二章 文化IP系统
  2.1 IP形象设计
  2.2 IP故事背景
  2.3 IP人格特征
  2.4 IP视觉系统
  2.5 IP延展应用

第三章 文化融入设计
  3.1 总体规划融入
      - 文化轴线设计
      - 空间序列营造
      - 场所精神塑造
  3.2 建筑设计融入
      - 地域风格建议
      - 文化符号应用
      - 材料工艺选择
  3.3 景观设计融入
      - 文化意境营造
      - 植物文化寓意
      - 文化小品设计
  3.4 室内设计融入
      - 空间氛围营造
      - 文化陈设艺术
      - 光影效果设计

第四章 文化体验设计
  4.1 文化动线规划
  4.2 体验节点设计
  4.3 高潮点营造
  4.4 五感体验设计
  4.5 仪式感营造

第五章 文化业态策划
  5.1 文化展示业态
  5.2 文化演艺业态
  5.3 文化商业业态
  5.4 文化活动策划

第六章 文化解说系统
  6.1 解说主题线索
  6.2 解说内容框架
  6.3 解说方式设计
  6.4 标识系统建议

第七章 实施建议
  7.1 分期实施建议
  7.2 重点打造区域
  7.3 文化合作资源
```

**参考团队**: 清华同衡/巅峰智业/德必集团
**交付物**:
- 《文化策划方案》（完整版，30-40页）
- IP形象设计图
- 文化融入设计建议书
- 文化体验动线图
- 文化业态落位图

#### 2.6 方案比选（投资+运营+文化专家参与评审）
- 多方案对比分析
- 投资收益对比
- 运营可行性评估
- **文化表达效果评估**
- 优缺点评估
- 推荐方案

**交付物**:
- 方案比选表（含投资、运营、文化维度）
- 推荐方案说明

---

### 阶段3: 方案设计（5-7天）

**输入**: 选定的概念方案 + 文化策划方案
**输出**: 《方案设计成果》+《文化策划深化方案》

**参与Agent**: 城市设计专家 + 建筑设计专家 + 投资金融专家 + 文旅运营专家 + **文化策划专家**

**工作内容**:

#### 3.1 详细投资测算（投资金融专家主导）
- 分年度投资计划
- 现金流预测
- 融资方案细化
- 退出路径设计（IPO/REITs/并购）
- 投资回报分析

**交付物**:
- 详细投资测算表
- 现金流预测表
- 融资方案书
- 投资回报分析

#### 3.2 运营策划深化（文旅运营专家主导）
- 招商策略（业态规划、品牌组合、租金策略）
- 运营机制（组织架构、服务流程、考核体系）
- 客户体验设计（动线、停留点、消费触发）
- 年度活动计划
- 会员体系设计

**交付物**:
- 招商策划方案
- 运营机制手册
- 客户体验设计
- 年度活动计划表

#### 3.3 详细规划（城市设计专家主导）
- 总平面深化
- 道路竖向设计
- 管线综合规划
- 指标核算

**参考团队**: 同济院/浙江省院
**交付物**:
- 详细总平面图
- 竖向设计图
- 管线综合图
- 经济技术指标表

#### 3.4 建筑设计（建筑设计专家主导，文化策划专家配合）
- 建筑单体方案
- 立面设计（文化策划专家：地域风格、文化符号）
- 剖面设计
- 典型平面
- **文化细节设计**（文化策划专家：细部符号、文化装饰）

**参考团队**: MAD/GOA
**交付物**:
- 建筑平面图
- 立面图
- 剖面图
- 效果图
- **文化细节设计图**

#### 3.5 文化策划深化（文化策划专家主导）
- **文化体验深化设计**: 具体场景、互动方式、技术实现
- **文化业态深化**: 具体品牌、面积、投资估算
- **文化活动策划深化**: 年度活动计划、IP活动设计
- **文化商业策划**: 文创产品、非遗市集、文化餐饮
- **文化解说系统深化**: 解说词、标识系统、智能导览

**新增交付物 - 《文化策划深化方案》**:
```
第一章 文化体验深化设计
  1.1 入口体验区
  1.2 文化展示区

  1.5 技术应用建议（AR/VR/光影）

第二章 文化业态深化
```

**交付物**:
- 景观平面图
- 节点效果图
- 植物配置表
- **文化小品设计图**

### 阶段4: 深化设计（7-14天）

**输入**: 确定的方案 + 文化策划深化方案
**输出**: 《深化设计成果》（施工图深度）+《文化落地实施手册》

**参与Agent**: 全体专家（含文化策划专家）

**工作内容**:

#### 4.1 规划深化（规划法规专家主导）
- 规划说明书
- 规划图纸（CAD）
- 规划模型（如需）

**交付物**:
- 规划设计说明书
- CAD图纸全套
- 规划模型文件

#### 4.2 投资深化报告（投资金融专家主导）
- 《可行性研究报告》（符合国家发改委格式）
- 《投资测算书》（详细财务模型）
- 《融资方案建议书》
- 《风险评估报告》

**格式标准**: 国家发改委《建设项目可行性研究报告编制指南》
**交付物**:
- 可行性研究报告（完整版）
- 投资测算Excel模型
- 融资方案书
- 风险评估报告

#### 4.3 运营策划手册（文旅运营专家主导）
- 《招商手册》（业态、品牌、政策）
- 《运营管理手册》（流程、标准、考核）
- 《开业筹备计划》（倒排工期、关键节点）
- 《年度运营计划》（活动、营销、目标）

**交付物**:
- 招商手册
- 运营管理手册
- 开业筹备计划
- 年度运营计划

#### 4.4 文化落地实施手册（文化策划专家主导）
- **文化施工图配合**: 与文化相关的建筑细节、景观小品、标识系统

## 四、自动化脚本库

### 4.1 批量处理脚本

**批量生成分析图**:
```python
#!/usr/bin/env python3
"""批量生成规划分析图"""

import matplotlib.pyplot as plt
import pandas as pd

def generate_land_use_chart(data, output_path):
    """生成用地平衡图"""
    fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 6))
    
    # 饼图
    ax1.pie(data['area'], labels=data['type'], autopct='%1.1f%%')
    ax1.set_title('用地面积占比')
    
    # 柱状图
    ax2.bar(data['type'], data['area'])
    ax2.set_title('用地面积对比')
    ax2.set_ylabel('面积(㎡)')
    
    plt.tight_layout()
    plt.savefig(output_path, dpi=300)
    plt.close()

# 使用示例
data = pd.DataFrame({
    'type': ['酒店', '住宅', '商业', '绿化'],
    'area': [25000, 40000, 15000, 20000]
})
generate_land_use_chart(data, 'land_use_analysis.png')
```
