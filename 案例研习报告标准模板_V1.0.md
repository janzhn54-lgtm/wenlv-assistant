# 案例研习报告标准模板 V1.0

## 📋 报告规格（爸爸确认版）

### 基本要求
- **文件大小**: 150KB+（内容充实）
- **页数**: 15-25页
- **格式**: PDF
- **图片**: 不包含（纯文字高质量分析）

### 报告结构（12个章节）

```
01 封面
    - 报告标题
    - 案例名称（中英文）
    - 10-Agent团队标识
    - 日期、工作室名称

02 目录
    - 12个章节列表

03 项目概况
    - 完整信息表格（15+项）
    - 项目定位与创新价值（2-3段）

04 规划法规专家分析
    - 用地性质与规划指标
    - 建筑退界与环保要求
    - 审批路径与合规要点

05 城市设计专家分析
    - 空间结构分析
    - 空间序列设计
    - 景观轴线与视线通廊

06 建筑设计专家分析
    - 建筑风格解析
    - 材料策略
    - 细部设计

07 投资金融专家分析
    - 投资结构分解
    - 收益模型（ADR/OCC/RevPAR）
    - 财务指标详表
    - 轻资产扩张模式

08 文旅运营专家分析
    - 运营定位
    - 运营数据详表
    - 竞争策略
    - 服务创新

09 文化策划专家分析
    - 文化主题定位
    - 文化IP开发

10 美术总监分析
    - 色彩策略
    - 光影设计

11 案例对比研究
    - 横向对比表格（3-6个竞品）
    - 差异分析

12 方法论提炼
    - 5个可复用方法论
    - 总结表格

13 附录
    - 参考资料
    - 相关案例推荐
```

### 排版规范

#### 字体大小
- 大标题: 24pt
- 章节标题: 14pt（深蓝色 #16213e）
- 小节标题: 11pt（深蓝 #0f3460）
- 正文: 9pt（黑色 #333333）
- 表格: 8.5pt
- 页脚: 8pt（灰色 #999999）

#### 段落格式
- 对齐: 左对齐
- 首行缩进: 18pt
- 行距: 14pt
- 段间距: 5pt

#### 表格样式
- 表头: 深蓝背景(#16213e) + 白色文字
- 表身: 浅灰背景(#f5f5f5) + 深色文字
- 边框: 0.5pt 浅灰
- 内边距: 4-5pt

#### 页面设置
- 页边距: 15mm（四边）
- 纸张: A4

### 内容深度要求

#### 每个Agent分析
- 字数: 300-500字/Agent
- 结构: 3-4个小节
- 数据: 具体数值+来源
- 深度: 不止描述，要有洞察

#### 数据表格
- 财务指标: 10+项
- 运营数据: 6+项
- 对比分析: 6+维度

### 质量标准

#### 必须包含
- [ ] 10个Agent全参与
- [ ] 3个以上数据表格
- [ ] 案例对比分析
- [ ] 方法论提炼
- [ ] 专业术语规范
- [ ] 排版美观统一

#### 禁止事项
- [ ] 不真实的图片
- [ ] 针对具体项目的建议（纯学习视角）
- [ ] 错别字
- [ ] 数据错误
- [ ] 逻辑矛盾

### 生成流程

1. **第1轮**: 生成完整内容框架
2. **第2轮**: 充实各Agent分析内容
3. **第3轮**: 添加数据表格
4. **第4轮**: 检查逻辑和数据
5. **第5轮**: 排版优化
6. **第6轮**: 最终润色

### 输出文件命名
```
{案例名称}_案例研习报告_{日期}.pdf
```

示例: `清迈四季酒店_案例研习报告_20260228.pdf`
