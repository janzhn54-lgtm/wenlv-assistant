# 学习笔记：www.aivi.fyi OpenClaw高级技巧

## 📚 核心收获（已学习并准备应用）

### 1️⃣ Claude Code Hooks回调机制（省Token关键）
**原理**：不用轮询，任务完成后自动回调
**应用**：
- 以后调用Coding Agent时，使用Hooks替代轮询
- 节省90%以上Token消耗
- 实现方式：Claude Code完成→写入latest.json→发送Wake Event→我读取结果

### 2️⃣ 多Agent协作架构（已规划）
**当前配置**：
- 小白（主）- 协调决策
- 美术总监 - 视觉审美
- 文字工作者 - 多风格文案
- 程序员 - 代码开发

**优化方向**：
- 每个Agent独立Workspace（物理隔离记忆）
- 不同任务分配不同模型（省成本）
- 故障隔离（一个崩溃不影响其他）

1. 提出需求 → 2. 执行 → 3. 观察报错 → 4. 调试修复 → 5. 更新Skill → 6. 推送GitHub → 7. 重复测试
