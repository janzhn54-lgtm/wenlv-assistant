## 🎨 设计语言体系

### 新中式风格
**核心逻辑**:
- 山水意境 → 现代演绎
- 院落空间 → 公共开放
- 材料质感 → 木石金属
- 色彩体系 → 黑白灰+木色

**代表案例**:
- 乌镇雅园（GOA）
- 杭州桃李春风（蓝城）
- 苏州桃花源（融创）

### 极简现代风格
**核心逻辑**:
- 形式追随功能
- 少即是多
- 纯净几何
- 光影变化

**代表案例**:
- 阿那亚（整体规划）
- 杭州天目里（伦佐·皮亚诺）
- 上海西岸美术馆（戴卫·奇普菲尔德）

### 生态自然风格
**核心逻辑**:
- 顺应地形
- 保护生态
- 低影响开发
- 海绵城市

**代表案例**:
- 安吉余村（两山理念）
- 成都麓湖生态城
- 杭州西溪湿地

## 📝 设计输出模板

### 概念设计阶段
1. **设计主题**（一句话）
2. **设计理念**（200字）
3. **空间结构**（一核两轴三片区）
4. **功能布局**（分区说明）
5. **景观策略**（绿色基础设施）
6. **设计亮点**（3-5个创新点）

### 方案深化阶段
1. **总平面图**（CAD+彩色）
2. **空间分析图**（图底、视线、流线）
3. **形态控制图**（高度、界面、节点）
4. **景观系统图**（绿地、水系、公共空间）
5. **效果图**（鸟瞰、人视）
