## 🏯 中国古建筑数据库

### 宋式建筑（典雅秀丽）
- **代表作**: 太原晋祠圣母殿
- **特征**: 屋檐起翘、斗拱硕大
- **比例**: 屋顶/屋身/台基 = 3:2:1
- **学习要点**: 《营造法式》

### 明清建筑（繁复严谨）
- **代表作**: 北京故宫、颐和园
- **特征**: 斗拱变小、装饰繁复
- **比例**: 严格的中轴对称
- **学习要点**: 《清式营造则例》

### 江南民居（灵动秀雅）
- **代表作**: 徽州民居、绍兴台门
- **特征**: 马头墙、青砖黛瓦
- **空间**: 四水归堂、天井采光
- **学习要点**: 水乡建筑特色

### 园林建筑（意境深远）
- **代表作**: 苏州园林、扬州园林
- **特征**: 虚实相生、移步换景
- **元素**: 亭、台、楼、阁、廊、舫
- **学习要点**: 《园冶》

## 🎨 设计手法库

### 屋顶设计
1. **坡屋顶现代转译** - 保留轮廓，简化构造
2. **平屋顶绿化** - 第五立面、生态屋顶
3. **复合屋顶** - 不同高度组合、层次丰富
4. **天窗采光** - 传统天井的现代演绎

### 立面设计
1. **格栅幕墙** - 传统窗棂的抽象化
2. **青砖砌筑** - 地方材料的当代表达
3. **木构装饰** - 结构外露、节点美学
4. **金属网帘** - 轻盈、半透明、光影

### 空间设计
1. **院落重组** - 传统四合院的现代演绎
2. **路径体验** - 游园式的空间序列
3. **视线控制** - 框景、借景、对景
4. **尺度转换** - 从公共到私密
