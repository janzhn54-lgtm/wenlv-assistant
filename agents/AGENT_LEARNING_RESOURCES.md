## 1️⃣ 规划法规专家 (Law-001)

### 必学法规
- [ ] 《浙江省国土空间规划条例》

- [ ] 《绍兴古城保护条例》

### 案例学习
- [ ] 每周收集5个浙江审批案例

### 学习方式
1. 每天访问官网查看新政策
2. 整理政策变化要点
3. 建立案例库（保存在 agents/planning-law-expert/cases/）
4. 每周总结学习笔记

## 📊 统一学习机制

### 每日学习任务
1. **上午**: 浏览专业网站，收集新资料
2. **下午**: 深度分析1-2个案例/文章
3. **晚上**: 整理学习笔记，更新记忆库

### 每周总结
- 整理本周学习内容
- 提炼核心知识点
- 更新案例库
- 输出学习周报

### 记忆存储位置
```
agents/
├── planning-law-expert/
│   ├── cases/              # 案例库
│   ├── regulations/        # 法规文件
│   └── learning-notes/     # 学习笔记
│
├── urban-design-expert/
│   ├── cases/              # 设计案例
│   ├── theories/           # 理论资料
│   └── design-patterns/    # 设计模式
│
├── [其他Agent同理]
```

### 汇报机制
- **每12小时**: Agent内部学习状态汇报
- **每周**: 向爸爸提交学习周报
- **每月**: 学习成果总结和能力评估
