# 实时监控方案 - 爸爸专用

## 📺 方案一：Web Dashboard（立即可用）

**地址**：http://10.0.1.20:18789

**功能**：
- 实时看到所有工具调用
- 查看会话历史
- 监控 token 消耗
- 查看运行状态

**缺点**：只能在 Codespace 内网访问，爸爸外面打不开

## 📱 方案二：飞书实时推送（推荐）

配置后，我的每个操作都会推送到爸爸的飞书：

### 配置步骤：

1. 爸爸在飞书群创建一个机器人
2. 获取 Webhook URL
3. 告诉我，我配置到系统

**效果**：
```
小白操作日志: 开始执行文件读取
小白操作日志: 执行 shell 命令: git status
小白操作日志: 完成代码生成，已保存
```

---

## 💻 方案三：浏览器同步（最直观）

在你的电脑上：

1. **安装 OpenClaw Chrome 扩展**
   - 去 Chrome 商店搜索 "OpenClaw"
   
2. **连接 Codespace**
   - 扩展设置里输入 Codespace 地址
   
3. **实时同步**
   - 我的操作实时显示在你的浏览器
   - 像看直播一样

---

## 📄 方案四：日志文件同步

我每操作一步，就写入共享日志：

```
~/.openclaw/logs/operations.log
```

爸爸可以通过以下方式查看：
- GitHub 网页直接查看
- 定时 cat 命令查看
- 配置自动推送

---

## 🎯 推荐组合

| 场景 | 推荐方案 |
|------|----------|
| 想实时看 | 飞书推送 + 浏览器同步 |
| 事后查 | 日志文件 + Web Dashboard |
| 手机看 | 飞书推送 |
| 电脑看 | 浏览器同步 |
