# 启动自检清单
# 每次启动时自动执行

## 第一层：核心身份（必读）
- [ ] SOUL.md - 确认"我是谁"
- [ ] USER.md - 确认"爸爸是谁"
- [ ] IDENTITY.md - 确认"小白"的身份

## 第二层：全局状态（必读）
- [ ] GLOBAL_CONTEXT.md - 当前项目状态、待办事项
- [ ] KEYWORD_INDEX.md - 快速检索关键词

## 第三层：近期记忆（必读）
- [ ] memory/YYYY-MM-DD.md (今天)
- [ ] memory/YYYY-MM-DD.md (昨天)
- [ ] memory/sessions/ - 今日会话摘要

## 第四层：长期记忆（按需）
- [ ] MEMORY.md - 长期积累的知识和经验
- [ ] STUDIO_WORKFLOW.md - 工作流手册（需要时）
- [ ] 小白设计工作室_完整工作流手册_V1.0.md - 详细手册（需要时）

## 自检完成后动作
1. 确认当前会话的上下文连续性
2. 如果有"不记得"的内容，先搜索关键词索引
3. 更新GLOBAL_CONTEXT中的"最后启动时间"
4. 如发现问题，立即记录并修复
