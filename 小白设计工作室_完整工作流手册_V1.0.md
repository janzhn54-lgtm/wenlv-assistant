## 🔄 6阶段工作流

```
┌─────────────────────────────────────────────────────────────────┐
│  阶段0      阶段1        阶段2        阶段3        阶段4        阶段5  │
│  启动  →   调研分析  →  概念设计  →  方案设计  →  深化设计  →  交付服务 │
│ 0.5天      2-3天       3-5天        5-7天       7-14天       持续   │
└─────────────────────────────────────────────────────────────────┘
```

## 阶段0: 项目启动（0.5天）

### 参与Agent
- 小白（项目经理）

### 核心工作
1. **需求澄清** - 3个问题确认法
2. **资料收集清单** - 明确需要爸爸提供的资料
3. **工作计划排期** - 甘特图制作
4. **团队分工确认** - 各Agent任务分配

### 输出文件
| 文件名称 | 格式 | 页数 |
|----------|------|------|
| 《项目任务书》 | Word | 1-2页 |
| 《资料需求清单》 | Excel | 1页 |
| 《项目甘特图》 | 图表 | 1页 |

---

## 阶段1: 调研分析（2-3天）

### 参与Agent
- 规划法规专家
- 投资金融专家
- 文旅运营专家
- 文化策划专家

### 工作模块与输出

#### 1.1 规划法规分析（规划法规专家主导）
**工作内容**:
- 上位规划解读
- 规划条件梳理
- 限制因素识别
- 审批路径规划

**输出文件**:
| 文件名称 | 格式 | 内容 |
|----------|------|------|
| 规划条件分析表 | Excel | 用地性质、指标、退界等 |
| 限制因素分析图 | PNG/PDF | 高压线、地铁、限高等 |
| 法规合规性评估 | Word | 合规项、风险提示 |

---

#### 1.2 市场分析 + 市场调研报告（文旅运营专家主导）
**工作内容**:
- 区位分析
- 竞品项目深度研究（运营模式、客群定位、收益结构）
- 客群定位（消费心理、行为模式）
- SWOT分析
- 运营前瞻性评估

**输出文件**:
| 文件名称 | 格式 | 页数 | 内容 |
|----------|------|------|------|
| 《市场调研报告》 | Word | 30-50页 | 宏观环境、行业分析、竞品研究、客群调研 |
| 区位分析图 | PNG | - | 交通、配套、竞争格局 |
| 竞品对比分析表 | Excel | - | 3-5个直接竞品详细对比 |
| 客群画像图谱 | PNG/PDF | - | 年龄、收入、偏好、行为 |

**报告结构**:
```
一、宏观市场环境
  1.1 政策环境分析
  1.2 经济环境影响
  1.3 社会文化趋势
  1.4 技术变革影响

二、行业市场分析
  2.1 文旅行业市场规模与增长
  2.2 细分市场分析（酒店/景区/商业）
  2.3 行业发展趋势
  2.4 新兴业态研究

三、区域市场研究
  3.1 绍兴市文旅市场概况
  3.2 越城区/迪荡湖板块分析
  3.3 供需关系分析
  3.4 价格水平研究

四、竞品项目调研
  4.1 直接竞品分析（3-5个）
  4.2 间接竞品分析
  4.3 标杆案例研究

五、客群需求调研
  5.1 目标客群画像
  5.2 消费行为分析
  5.3 需求痛点识别
  5.4 支付意愿研究

六、市场机会与风险评估

七、结论与建议
```

---

| 风险提示清单 | Word | 政策/市场/运营/财务风险 |

---

#### 1.4 基地分析（规划法规专家主导）
**工作内容**:
- 现状测绘分析
- 交通可达性
- 景观资源评估

**输出文件**:

|----------|------|------|

---

#### 1.5 文化资源调研（文化策划专家主导）
**工作内容**:
- 地域文化研究（历史沿革、文化脉络、精神内核）
- 物质文化调查（古建筑、文化景观）
- 非物质文化挖掘（民俗、技艺、节庆、传说）
- 周边文化资源（项目周边文化景点、文化设施）
- 文化价值评估（独特性、影响力、活化潜力）

**输出文件**:
| 文件名称 | 格式 | 页数 | 内容 |
|----------|------|------|------|
| 《文化资源调查报告》 | Word | 20-30页 | 区域文化、物质/非物质文化、周边资源、价值评估 |
| 文化资源分布图 | PNG/PDF | - | 文化资源空间分布 |
| 文化价值评估表 | Excel | - | 各资源独特性、影响力评分 |
| 可利用文化元素清单 | Word | - | 符号、故事、IP开发建议 |

**报告结构**:
```
第一章 区域文化背景
  1.1 历史文化沿革
  1.2 地域文化特征
  1.3 文化精神内核

第二章 物质文化资源
  2.1 历史建筑遗存
  2.2 文化景观资源
  2.3 工业遗产资源（如适用）
  2.4 保护状态评估

第三章 非物质文化资源
  3.1 民俗文化
  3.2 传统技艺
  3.3 地方戏曲/曲艺
  3.4 节庆活动
  3.5 民间传说/故事

第四章 周边文化环境
  4.1 周边文化景点
  4.2 文化设施分布
  4.3 文化产业发展
  4.4 文化氛围评估

第五章 文化价值评估
  5.1 独特性评估
  5.2 影响力评估
  5.3 活化潜力评估
  5.4 与项目契合度分析

第六章 可利用文化元素
  6.1 核心文化符号
  6.2 文化故事素材
  6.3 文化体验素材
  6.4 IP开发建议

第七章 结论与建议
  7.1 文化定位建议
  7.2 文化融入策略
  7.3 文化业态建议
```

## 阶段2: 概念设计（3-5天）

### 参与Agent
- 城市设计专家
- 投资金融专家
- 文旅运营专家
- 建筑设计专家
- 文化策划专家

### 工作模块与输出

#### 2.1 投资测算深化（投资金融专家主导）
**工作内容**:
- 详细投资估算（土地、建安、前期、财务费用）
- 收益预测模型（分业态、分年度）
- 财务指标计算（NPV、IRR、回收期）
- 敏感性分析
- 融资方案设计（股权、债权、REITs等）

**输出文件**:
| 文件名称 | 格式 | 内容 |
|----------|------|------|
| 投资测算表 | Excel | 完整财务模型 |
| 收益预测报告 | Word | 分年度收入、成本、利润 |
| 融资方案建议书 | Word | 多渠道融资对比 |

---

#### 2.2 运营策划前置（文旅运营专家主导）
**工作内容**:
- 商业定位与业态策划
- 客群细分与产品匹配
- 运营模式设计（自营/租赁/合作）
- 收入结构规划
- 关键运营指标设定

**输出文件**:
| 文件名称 | 格式 | 内容 |
|----------|------|------|
| 运营策划方案 | Word | 定位、模式、收入结构 |
| 业态配比建议 | Excel | 各业态面积、比例 |
| 收入结构预测 | Excel | 门票/餐饮/住宿/商业占比 |

---

#### 2.3 概念生成（城市设计专家主导）
**工作内容**:
- 案例研究（国内外，重点分析运营成功案例）
- 设计理念提炼
- 主题定位
- 功能策划（与运营策划联动）

**输出文件**:
| 文件名称 | 格式 | 内容 |
|----------|------|------|
| 案例研究报告 | Word | 3-5个对标案例深度分析 |
| 设计理念阐述 | Word | 设计概念、主题定位 |
| 功能配比表 | Excel | 酒店/住宅/商业面积配比 |

---

#### 2.4 空间规划（城市设计专家主导，文化策划专家配合）
**工作内容**:
- 总体布局（3-5个方案）
- 功能分区（与业态规划、文化策划联动）
- 交通组织
- 景观结构
- **文化空间序列**（文化策划专家：文化轴线、场所精神、文化节点）

**输出文件**:
| 文件名称 | 格式 | 内容 |
|----------|------|------|
| 总平面图 | CAD+PNG | 3-5套方案 |
| 功能分区图 | PNG | 各功能板块划分 |
| 交通分析图 | PNG | 车流动线、人流动线 |
| 空间结构图 | PNG | 空间序列、景观结构 |
| 文化空间规划图 | PNG | 文化轴线、节点、场所精神 |

---

#### 2.5 文化策划融入设计（文化策划专家主导）
**工作内容**:
- 文化主题提炼（基于调研确定核心文化主题）
- 文化IP策划（IP形象、故事线、视觉系统）
- 文化融入策略（如何在空间、建筑、景观中表达文化）
- 文化体验设计（文化动线、体验节点、互动方式）
- 文化业态建议（文化展示、演艺、商业、活动）

**输出文件**:
| 文件名称 | 格式 | 页数 | 内容 |
|----------|------|------|------|
| 《文化策划方案》 | Word | 30-40页 | 主题定位、IP系统、融入策略、体验设计、业态策划 |
| IP形象设计图 | PNG/PDF | - | 角色形象、视觉系统 |
| 文化融入设计建议书 | Word | - | 总体规划、建筑、景观、室内融入建议 |
| 文化体验动线图 | PNG | - | 参观动线、体验节点 |
| 文化业态落位图 | PNG | - | 各文化业态空间落位 |

**报告结构**:
```
第一章 文化主题定位
  1.1 核心文化主题
  1.2 主题演绎框架
  1.3 文化定位语
  1.4 文化愿景

第二章 文化IP系统
  2.1 IP形象设计
  2.2 IP故事背景
  2.3 IP人格特征
  2.4 IP视觉系统
  2.5 IP延展应用

第三章 文化融入设计
  3.1 总体规划融入
      - 文化轴线设计
      - 空间序列营造
      - 场所精神塑造
  3.2 建筑设计融入
      - 地域风格建议
      - 文化符号应用
      - 材料工艺选择
  3.3 景观设计融入
      - 文化意境营造
      - 植物文化寓意
      - 文化小品设计
  3.4 室内设计融入
      - 空间氛围营造
      - 文化陈设艺术
      - 光影效果设计

第四章 文化体验设计
  4.1 文化动线规划
  4.2 体验节点设计
  4.3 高潮点营造
  4.4 五感体验设计
  4.5 仪式感营造

第五章 文化业态策划
  5.1 文化展示业态
  5.2 文化演艺业态
  5.3 文化商业业态
  5.4 文化活动策划

第六章 文化解说系统
  6.1 解说主题线索
  6.2 解说内容框架
  6.3 解说方式设计
  6.4 标识系统建议

第七章 实施建议
  7.1 分期实施建议
  7.2 重点打造区域
  7.3 文化合作资源
```
